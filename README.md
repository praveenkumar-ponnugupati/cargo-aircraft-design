# 80-Ton Cargo Aircraft Design Project

**B.Tech Final Year Project — Aerospace Engineering**
Bharath Institute of Higher Education and Research, Chennai | 2021

---

## Overview

This project covers the complete conceptual design of an 80-ton payload cargo aircraft from scratch — from engine selection and airfoil analysis to structural stability and centre of gravity verification. Every major parameter was computed iteratively using engineering formulas, Java Foil, CATIA V5, and Python-based automation.

The goal: prove the aircraft is aerodynamically and structurally stable before anything is physically built.

---

## What We Designed

| Parameter | Value |
|---|---|
| Payload | 80 tonnes |
| Engine | Pratt & Whitney (x4), 190 kN thrust each |
| Total Thrust | 760 kN |
| Airfoil | NACA 63A-910 |
| Aircraft Length | 67 metres |
| Centre of Gravity | 47% of fuselage length |
| Wing Design Tool | CATIA V5 |
| Boundary Layer | Dominant turbulent flow (high Reynolds number) |

---

## Computational Approach

Parameter optimisation was automated using Python scripts. Predictive analytics and clustering were applied to identify stable design configurations, achieving a 50% improvement in computational accuracy.

---

## Tools Used

- Java Foil — airfoil analysis and selection
- CATIA V5 — 3D wing and control surface design
- ANSYS — structural and aerodynamic verification
- Python — automation of design parameter calculations

---

## Stability Conclusion

After verifying all major parameters, the design was confirmed aerodynamically and structurally stable.
